# 🎬 Sistema de Locadora - Exercício de POO em Python

Este projeto foi desenvolvido como parte dos meus estudos no curso de **Análise e Desenvolvimento de Sistemas (ADS)**. O objetivo foi praticar os fundamentos da **Programação Orientada a Objetos (POO)** utilizando Python.

## 🚀 Conceitos Praticados

Neste código, explorei os seguintes pilares e recursos:

* **Classes e Objetos:** Criação do molde `Filme` e instanciamento de objetos reais.
* **Método `__init__`:** Inicialização de atributos como título, gênero e status.
* **Atributos Dinâmicos:** Criação de estados para o objeto (Disponível/Alugado).
* **Métodos de Classe:** Implementação de comportamentos para `alugar()`, `devolver()` e `exibir_info()`.
* **Boas Práticas (PEP 8):** Organização do código seguindo as regras de espaçamento e nomenclatura.

## 🛠️ Tecnologias Utilizadas

* **Python 3**
* **VS Code** (com extensões Pylance e Flake8 para garantir a qualidade do código)
