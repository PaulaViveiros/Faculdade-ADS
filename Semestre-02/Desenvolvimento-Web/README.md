# 🐔 Projeto Galo Chef - Estudos de Flask

Este repositório faz parte da minha jornada no curso de **Análise e Desenvolvimento de Sistemas (ADS)**, com foco em desenvolvimento Back-end utilizando Python.

## 🚀 O que este projeto faz?
Este é um servidor web funcional criado com o framework **Flask**. Ele demonstra o conceito de rotas dinâmicas e o funcionamento de um servidor local no ambiente de desenvolvimento.

## 🛠️ Tecnologias Utilizadas
* **Python**: Linguagem principal.
* **Flask**: Micro-framework para desenvolvimento web.
* **Git/GitHub**: Controle de versão e portfólio.

## 📖 O que aprendi nesta etapa:
- Instalação e gerenciamento de pacotes com `pip`.
- Criação e configuração de rotas (`@app.route`).
- Resolução de conflitos de diretórios e arquivos duplicados.
- Geração automática de dependências com `requirements.txt`.

## ⚙️ Como rodar o projeto
1. Instale as dependências: `pip install -r requirements.txt`
2. Execute o servidor: `py app.py`
3. Acesse no navegador: `http://127.0.0.1:5000`
